PROMPT DE COMANDO

*sempre em ingles

comandos no prompt de comando:

"cd (nome da pasta ou o caminho da pasta desejada)"-> abrir a pasta

"cd.." -> fecha a pasta

"cd\" -> fecha todas as pastas abertas

"cls" -> limpa a tela

"dir" -> visualizar o conteúdo de um diretório

"md (nome da pasta)" -> comando para criar uma pasta

"git add ." -> adicionar itens na memória

"git commit -m "Arquivo criado"" -> cria um rótulo na memória

"git push" -> grava o que está na memoria no github.

"git config --global user.email "(email conta git hub)""

"git config --global user.name "(nome usuario git hub)""
